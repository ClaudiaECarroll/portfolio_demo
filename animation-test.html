Squirrel Animation Test Page

This is a test page for the squirrel and robot chase animation. The animation should appear at the bottom of the screen with the following sequence:

  1. Left door appears and opens (0-1s)
  2. Squirrel runs out to halfway point between doors (1-4s)
  3. Squirrel stops and looks back (4-5s)
  4. Robot emerges from left door (4.5s)
  5. Right door appears and opens (5s)
  6. Squirrel continues running to exit door (5.5-8.5s)
  7. Robot follows squirrel into exit door
  8. Both doors close and disappear (8.5-10s)

The squirrel should stop exactly halfway between the two doors. Use the controls in the top-right to test the animation manually.
